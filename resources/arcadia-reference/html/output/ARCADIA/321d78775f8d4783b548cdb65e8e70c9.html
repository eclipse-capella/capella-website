Engineering interaction Contents

Major Kinds of Engineering Data > SOLUTION Description > LOGICAL ARCHITECTURE > System Capabilities in LA

A capability is described by a set of scenarios, and functional chains (or operational processes in operational analysis), each describing a use case for the capability.

Involved Engineering Data

  • scenarios realising the capability : Notional Scenarios
  • capabilities achieved by using the system : Notional Solution Capabilities
  • functional chains realising the capability : Notional Functional Chains
