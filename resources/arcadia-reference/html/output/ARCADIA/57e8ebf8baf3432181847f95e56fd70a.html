Major Kinds of Engineering Data > SOLUTION Description > LOGICAL ARCHITECTURE > Components Scenario in LA

A scenario describes a dynamic, time-related interaction between the system or its components and actors.

This interaction is expressed through the sequence of exchanges between these elements, as described in the dataflow. These exchanges are temporally ordered.

 

 

 

 

Involved Engineering Data

  • components whose exchanges are described in the scenario : Notional Components performing Functions
  • time-ordered sequence of exchanges between components and actors : Notional Comp. Scenarios
  • actors involved in the scenario : External systems/actors in LA
  • exchanges in a time-related order : Notional Components Exchanges
