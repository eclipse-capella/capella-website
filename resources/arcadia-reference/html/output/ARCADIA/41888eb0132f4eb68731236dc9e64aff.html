Scenarios involving Element

Engineering Data

Major Kinds of Engineering Data > SOLUTION ELEMENT NEED description > Scenarios involving Element

A specified scenario describes a usage of the element in a given context. It is a time-ordered set of functional exchanges between functions, or between the element and external actors or elements, to fulfil a Capability.

Activities producing the Data

  • Define & enforce a PBS and Component Integration Contract
  • Formalise Component development & integration contract

Activities using the Data

  • Build PIDS documents (Prime Item Development Specification) and subsystems, SW, HW specifications (SSS, SRS, HRS) including interfaces
  • Perform SW engineering

Usages

  • Software Requirements (SRS or equivalent in Agile)
