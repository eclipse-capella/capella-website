ARCADIA Engineering Landscape User Guide Reference Manual


SOLUTION ELEMENT NEED description

Set grouping various elements (see Content below)

This data set groups data and documents related to the definition of expectations on a system element, based on the former system solution definition.

Content

  • Solution Element Need Specification
  • Solution Element
  • External systems or other Elements
  • Element Capabilities
  • Functional Chains involving Element
  • Scenarios involving Element
  • Functions expected from Element
  • Element Functional Exchanges
  • Element Exchanged Items
  • Element Data
  • Element Modes & States
  • Element external Components Exchanges
  • Element external Interfaces
  • Element/neighbourhood Scenarios
  • Element/neighbourhood Physical Links
  • Element Textual Requirements

Applied Constraints

Name and Description Owned specification Constrained elements
Solution Element Need is a subset of System Solution Description ; it is generated by extracting only those elements that are of intrerest for solution element engineering. This Solution element need should not be modified, but rather an update of solution description by a co-engineering work
