Major Kinds of Engineering Data > SOLUTION Description > LOGICAL ARCHITECTURE > Notional Comp. Scenarios

A component scenario describes the behavior of the system in a given context. It is a time-ordered set of exchanges between components, or between the system and external actors, to fulfil a Capability.
