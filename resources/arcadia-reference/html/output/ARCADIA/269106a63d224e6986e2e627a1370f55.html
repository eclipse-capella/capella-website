Scenarios in SA

Engineering interaction Contents

Major Kinds of Engineering Data > NEED Description > SYSTEM NEED > Scenarios in SA

A scenario describes a dynamic, time-related interaction between the system or its components and actors.

This interaction is expressed through the sequence of exchanges between these elements, as described in the dataflow. These exchanges are temporally ordered.

 

 

 

 

Involved Engineering Data

  • time-ordered sequence of exchanges between system, actors or between functions : Specified Scenarios
  • exchanges in a time-related order : Spec. Functional Exchanges
  • functions whose exchanges are described in the scenario : Specified Functions
  • actors involved in the scenario : External systems/actors in SA
  • the system : System
