Functional exchanges FIP 2_1 Define interfaces and exchanges between behavioural components

Interactions or exchanges between functions are formalised through functional exchanges.

Each exchange links one function output to one function input. These inputs/outputs are formalised as function ports; the set of ports allocated to a function caracterises the way it should be used.

The description of functions and exchanges linking them is known as functional dataflow.

The nature of what is exchanged between two functions is expressed as one (or several) exchange items. Each of them is linked to one or several exchanges carrying them. An exchange item references data of the global data model.
