Component exchanges FOP 1 Define interfaces and exchanges between behavioural components

Interactions or exchanges between behavioural components are formalised through component exchanges.

Each exchange links one component output to one component input. These inputs/outputs are formalised as component ports; the set of ports allocated to a component caracterises the way it should be used.

Each component port is associated to interfaces describing its ability to interact with other components.

Inside each interface, the nature of what is exchanged with other components is expressed as one (or several) exchange items. An exchange item references data of the global data model.

Components items such as exchanges, ports and interfaces are defined and justified by referencing corresponding functional exchanges, function ports and exchange items, usually through grouping functional items into component items.
