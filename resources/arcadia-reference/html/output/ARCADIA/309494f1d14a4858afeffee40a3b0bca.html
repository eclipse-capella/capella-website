A mode or state characterizes the system behaviour in given environmental conditions. A mode or state is activated by external conditions and/or by system users.

Only some paths (named transitions) are allowed between modes or states, usually under internal or external conditions.

A mode or state may specify which functions are available or not during this mode or state.
