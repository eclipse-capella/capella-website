Engineering activities > Engineering Activities & Perspectives > DESIGN THE SOLUTION ARCHITECTURE > Design LOGICAL ARCHITECTURE > Build candidate architectural breakdowns in Components > Components

A logical component is a part of a notional breakdown of the system, a first sketch of architecture. This kind of component is usually independent of technology choices.

Several components may be grouped, for complexity management and separation of concerns purpose, into one father component, so as to manage complexity and hide details inside the component. This leads to a component tree structuring the components(also known as component breakdown).

Source activity

  • Define a system breakdown into logical Components

Destination activity

  • Allocate functions to components
