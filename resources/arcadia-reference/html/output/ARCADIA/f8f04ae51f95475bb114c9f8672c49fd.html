Engineering interaction Contents

Major Kinds of Engineering Data > SOLUTION Description > PHYSICAL ARCHITECTURE > Components Scenario in PA

A component scenario  describes the behavior of the system in a given context. It is a time-ordered set of exchanges between components, or between the system and external actors, to fulfil a Capability.

Involved Engineering Data

  • actors involved in the scenario : External systems/actors in PA
  • components whose exchanges are described in the scenario : Components performing Functions
  • time-ordered sequence of exchanges between components and actors : Design Comp. Scenarios
  • exchanges in a time-related order : Components Exchanges
