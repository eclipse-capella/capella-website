Design Comp. Scenarios

Engineering Data

Major Kinds of Engineering Data > SOLUTION Description > PHYSICAL ARCHITECTURE > Design Comp. Scenarios

A component scenario describes the behavior of the system in a given context. It is a time-ordered set of exchanges between components, or between the system and external actors, to fulfil a Capability.

Activities producing the Data

  • Deduce scenario allocation to components
  • Select and finalise the Physical Reference Architecture

Activities using the Data

  • Build ICD document (Interface Control Document)
  • Check each engineering/architecting decision against physical viewpoints

Usages
