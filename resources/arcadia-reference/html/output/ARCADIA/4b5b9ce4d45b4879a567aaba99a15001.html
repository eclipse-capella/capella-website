Enabling/Test Means Definition

Engineering Data

Major Kinds of Engineering Data > IVV > IVV MASTERING > Enabling/Test Means Definition

The definition of elements not included into the set of system elements to be tested, that are necessary in order to test system elements. They can be external systems, benches, simulation software, stimulation and outputs analysis devices, and also system elements already verified/validated, etc.
