ARCADIA Engineering Landscape User Guide Reference Manual


System

Engineering Data

Major Kinds of Engineering Data > NEED Description > SYSTEM NEED > System

The system of interest or solution to be delivered to customers(to be considered as a generic name: could be a software, a service, etc.).

Activities producing the Data

  • Complement and detail internal functional Analysis
  • Allocate functions to the system and to users
  • Define functional Scenarios
  • Define System Modes & States
  • Allocate functions to components
  • Perform a functional and non-functional Analysis

Activities using the Data

  • Define interfaces and exchanges between behavioural components
  • Build IRS document (Interface Requirements Specification)
  • Implement behavioural components on implementation components
  • Validate functional Analysis
  • Define functional Scenarios
  • Define System Modes & States

Usages

  • Scenarios in SA
  • Modes & states in SA
  • Allocated functions in SA
