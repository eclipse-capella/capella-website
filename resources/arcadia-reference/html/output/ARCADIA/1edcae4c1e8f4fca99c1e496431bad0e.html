Scenarios in LA

Engineering interaction Contents

Major Kinds of Engineering Data > SOLUTION Description > LOGICAL ARCHITECTURE > Scenarios in LA

A scenario describes a dynamic, time-related interaction between the system or its components and actors.

This interaction is expressed through the sequence of exchanges between these elements, as described in the dataflow. These exchanges are temporally ordered.

 

 

 

 

Involved Engineering Data

  • time-ordered sequence of exchanges between system, actors or between functions : Notional Scenarios
  • exchanges in a time-related order : Notional Functional exchanges
  • functions whose exchanges are described in the scenario : Notional Functions
  • actors involved in the scenario : External systems/actors in LA
  • components involved in the scenario : Notional Components performing Functions
